# REIT / Property Method Notes

## REITs

- Primary anchor: `NAV`
- Secondary anchor: `AFFO` multiple
- Focus on stabilized NOI, cap rates, occupancy, leverage, and near-term refinancing

## Property Platforms

- Use this pack when the asset base and recurring property cash flows explain value better than a
  generic operating-company DCF.
- Be explicit about platform or development value that sits outside stabilized rental NOI.

## Developers With Stabilized Assets

- Split stabilized assets from development pipeline.
- Use explicit haircuts when development value is uncertain or far-dated.

## Pack Selection Rule

Use this extension pack when:

- property-level NOI and cap rates are first-class valuation inputs
- `AFFO` or `FFO` is a meaningful market anchor
- development and stabilized assets need separate valuation treatment
