# Financials Method Notes

## Banks

- Primary anchor: `P/TBV` or `P/B` tied to sustainable `ROTE` / `ROE`
- Secondary anchor: residual income
- Focus on tangible book, capital adequacy, credit costs, and normalized returns

## Insurers

- Primary anchor: `P/B` tied to sustainable `ROE`
- Secondary anchor: residual income or embedded value
- Focus on reserve adequacy, investment yield, and combined ratio or new-business quality

## Specialty Finance

- Use this pack only if book value and credit economics are the main valuation anchor.
- If value is still better explained by operating cash flow, stay in core `valuation-ultra`.

## Pack Selection Rule

Use this extension pack when:

- book value is a meaningful valuation anchor
- returns on equity are more important than unlevered free cash flow
- capital ratios or reserve quality are first-class valuation inputs
