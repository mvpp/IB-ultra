# QC And Output

## Required Tabs / Deliverables

Minimum package:

- `Valuation_Prep`
- `Capital_Cost`
- `DCF` or primary-method tab
- `Comps`
- `Scenarios`
- `Football_Field`
- `Target_Price`
- `Valuation_QC`

## Capital Cost Checks

- Cost of equity inputs are explicit: risk-free rate, beta, ERP, country premium, size premium if used
- Cost of debt is tied to actual debt economics or a defendable synthetic rate
- Target leverage is stated and justified
- WACC matches method framing

## DCF Checks

- Terminal growth < long-run nominal GDP unless clearly justified
- Terminal margin is consistent with competitive structure
- Terminal reinvestment is consistent with terminal growth and ROIC
- Implied terminal ROIC is not nonsensical
- Enterprise value equals PV of forecast cash flows + PV of terminal value

## Comps Checks

- Peer set documented
- Outliers flagged
- Metric definition consistent across peers
- Fiscal periods aligned
- Current trading date and market inputs dated

## EV Bridge Checks

- Enterprise value to equity value reconciles exactly
- Cash treatment explicit
- Lease liabilities explicit
- Minority interest explicit
- investments and associates explicit
- diluted shares explicit

## Scenario Checks

Minimum:

- base / bull / bear
- one operating sensitivity table
- one valuation sensitivity table
- reverse DCF or market-implied expectation output

For binary-risk names:

- probability-weighted scenario output

## Final Output Standard

`Target_Price` should include:

1. Primary method result
2. Secondary method result
3. Method weighting
4. Implied equity value
5. Target price
6. Current price
7. Upside / downside
8. Key assumptions
9. Principal risks
10. Monitoring points

## Final Validation Gate

Do not finalize unless all of the following are true:

- primary and secondary methods both completed
- EV bridge ties
- diluted shares tie
- sensitivities run
- reverse DCF or implied expectations run
- no unresolved accounting normalization issue is hiding inside the bridge
