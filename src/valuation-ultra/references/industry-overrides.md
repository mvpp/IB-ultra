# Industry Overrides

Read this file before building the primary method if the company is in one of these groups.

## Banks

- Use `P/B` tied to sustainable `ROE` / `ROTCE`.
- Use residual income as the secondary method.
- Focus on CET1, NIM, credit costs, and normalized returns.
- Do not force a standard unlevered DCF unless there is a very specific reason.

## Insurance

- Use `P/B` tied to sustainable `ROE`.
- For life insurers, add embedded value logic where disclosure allows.
- Normalize investment yield and combined ratio.

## REITs / Property

- Use NAV and AFFO.
- Separate development pipeline from stabilized assets.
- Make cap-rate assumptions explicit and sensitivity-tested.
- Treat recurring maintenance capex carefully.

## Utilities / Regulated Assets

- Use RAB or DDM as primary.
- Tie value to allowed return, regulated base growth, and leverage capacity.
- WACC and allowed ROE comparison must be explicit.

## E&P / Mining

- Use NAV on reserves / assets.
- Use commodity decks and sensitivity tables.
- Add EV/EBITDA or P/NAV as a secondary market check.

## Biotech / Binary Clinical

- Use rNPV with explicit probability by asset and stage.
- Add cash runway and expected dilution bridge.
- Probability, launch timing, and peak sales are first-class assumptions.

## Conglomerates / Multi-Business Mix

- Use SOTP.
- Value each business with its own method family.
- Add holding-company discount only with explicit rationale.
